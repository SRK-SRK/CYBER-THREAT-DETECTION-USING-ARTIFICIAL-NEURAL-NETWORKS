🚀 Features

GUI Application (Tkinter) – easy-to-use interface for dataset upload & model selection

Preprocessing with TF-IDF for feature extraction from event profiles

Event Vector Generation for structured training & testing

Multiple Algorithms Implemented:

LSTM (Deep Learning)

CNN (Deep Learning)

SVM

KNN

Random Forest

Decision Tree

Naive Bayes

SVM + PSO (extension with feature optimization)

Performance Metrics: Accuracy, Precision, Recall, F-measure

Graphical Comparison: Accuracy, Precision, Recall, F-Measure comparison plots




🛠️ Tech Stack

Programming Language: Python

GUI Framework: Tkinter



Libraries:

Numpy, Pandas, Scikit-learn

Keras (TensorFlow backend)

Matplotlib

Pyswarms (for PSO)



📂 Project Structure
├── datasets/                 # Place your training dataset here
├── cyber_threat.py       # Main application file (Tkinter GUI + ML models)

Upload dataset

Preprocess with TF-IDF

🚀 How to Run


Install dependencies:

pip install numpy pandas scikit-learn matplotlib keras tensorflow pyswarms


Run the application:

python cyberthreatdetection.py
